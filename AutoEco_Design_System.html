7. Иконки

Библиотека иконок и правила использования

Иконографика: Lucide Icons (stroke-based, 24×24 grid). В интерфейсе используются размеры:

• w-3.5 h-3.5 (14px) — внутри тегов, мелкие детали
• w-4 h-4 (16px) — навигация, кнопки, инлайновые иконки
• w-5 h-5 (20px) — карточки, заголовки секций
• w-10 h-10 (40px) — аватар, крупные плейсхолдеры
• w-12 h-12 (48px) — пустые состояния (empty state)

Цвет иконок наследуется от текста (currentColor). Для акцентных иконок — text-brand-light (#7B5CF0).

9. Принципы дизайна

Руководящие принципы при создании новых экранов и компонентов

🧊
Light Glassmorphism
Полупрозрачные белые поверхности с blur. Хедер: white/92% + blur(20px). Чистый, воздушный интерфейс.
🎯
Минимальные границы
Границы — prussian/4-8%. Разделение через тени и пространство, не через жёсткие линии.
💜
Фиолетовый — фокус
Brand purple только для CTA и активных состояний. Не перенасыщать — использовать дозированно.
📐
Крупные скругления
Radius 14-24px для основных элементов. Мягкие формы, никаких острых углов в UI.
