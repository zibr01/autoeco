# AutoEco — Кликабельный прототип

## Запуск

### 1. Установи Node.js
Скачай и установи с [nodejs.org](https://nodejs.org) (версия 18+, LTS)

### 2. Установи зависимости
```bash
cd /Users/kirill/Documents/Экосистема/autoeco
npm install
```

### 3. Запусти
```bash
npm run dev
```

Открой в браузере: **http://localhost:3000**

---

## Страницы прототипа

| URL | Описание |
|-----|----------|
| `/` | Лендинг — главная страница |
| `/garage` | Мой гараж — список 3 автомобилей |
| `/garage/1` | BMW 5 Series — профиль с ТО, напоминаниями |
| `/garage/2` | Toyota Camry — есть просроченное ТО |
| `/garage/3` | Lada Vesta — новая, состояние отличное |
| `/services` | Каталог сервисов (8 шт.) с фильтрами |
| `/services/1` | АвтоТех Премиум — онлайн-запись работает |
| `/diagnostics` | AI-диагностика — симулированный чат |
| `/parts` | Подбор запчастей по VIN |

## Деплой на Vercel (бесплатно)
